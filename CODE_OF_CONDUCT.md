Code of Conduct
Our Commitment

The Nodus project is committed to providing a respectful and constructive environment for contributors and users.

We want the community around Nodus to focus on:

Learning

Collaboration

Language design exploration

Building useful tools

Everyone participating in this project is expected to follow this code of conduct.

Expected Behavior

Participants should:

Be respectful and constructive in communication

Focus on technical discussion rather than personal criticism

Provide helpful feedback on code and ideas

Assume good intent from other contributors

Healthy technical disagreement is encouraged, but it must remain professional.

Unacceptable Behavior

The following behaviors are not acceptable in the Nodus project:

Personal attacks or harassment

Insults, hostile language, or intimidation

Discriminatory remarks

Deliberately disruptive behavior in issues or pull requests

Publishing private information without consent

Enforcement

Project maintainers have the right to:

Remove comments or contributions that violate this policy

Close issues or pull requests that become disruptive

Restrict or block contributors who repeatedly violate the code of conduct

These decisions will be made to preserve the health of the project.

Scope

This code of conduct applies to:

GitHub issues

Pull requests

Discussions

Any official Nodus community spaces

Final Note

Nodus is an experimental language project intended for learning, exploration, and disciplined engineering practice.

Constructive collaboration helps everyone improve.
