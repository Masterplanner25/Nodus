# Nodus 0.2.0 Release Notes

Nodus 0.2.0 is the first polished release focused on real-world scripting. It delivers a stable module system, a coherent standard library, and a disciplined toolchain for formatting, validation, and CI.

Highlights:
- Module system with imports/exports, re-exports, and deterministic resolution.
- Standard library modules (`std:strings`, `std:collections`, `std:fs`, `std:path`).
- `nodus fmt` and `nodus check` for reliable development workflows.
- Debugging tools (`--dump-bytecode`, `--trace` with controls).
- CI workflow and release discipline docs.

If you're new to Nodus, start with `GETTING_STARTED.md` and run `nodus run examples/hello.nd`.
